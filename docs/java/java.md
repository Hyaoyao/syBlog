### 数据类型

**计算机存储单元：**计算机是可以用来存储数据的，但是无论是内存还是硬盘，计算机存储设备最小的信息单位叫=="位（bit）"==，又称之为“比特位”，通常用小写字母==“b”==来表示，而计算机中最小的存储单元叫==“字节（byte）”==，通常用大写字母=="B"==表示，字节是由连续8个位组成。

常用的储存单位：

1B(字节) = 8bit

1KB = 1024B

1MB = 1024KB

1GB = 1024MB

1TB = 1024GB



**数据类型：**Java语言是强类型，对于每一种数据都给出明确的的数据类型，不同的==数据类型==也分配了不同的==内存空间==，所以他们表示的==数据大小==也不一样。

java的数据类型主要可以分为两种：基本数据类型、引用数据类型

+ 基本数据类型

  数值型：==整数（byte，short，int，long）==，==浮点数（float，double）==，==字符（char）==

  非数值型：==布尔（boolean）==

+ 引用数据类型

  ==类（class）==，==接口（interface）==、==数组（[]）==

## 标识符

标识符：是给类、方法、变量等方法起名的符号。

**定义规则**

+ 由数字、字母、下划线（_）和美元符（$）组成
+ 不能以数字开头
+ 不能是关键字
+ 区分大小写

**常见的命名约定**

小驼峰命名法：方法、变量

+ 约定1：标识符是一个单词的时候，首字母是小写
+ 范例：name
+ 约定2：标识符由多个单词组成的时候，第一个单词的首字母小写，其他单词字母为大写开头
+ 范例2：helloWord



**大驼峰命名法：类**

+ 约定1：标识符是一个单词的时候，首字母为大写
+ 范例1：Student
+ 约定2：标识符为多个单词组成的时候，每个单词的首字母为大写。
+ 范例2：GoodStudent

#### 逻辑运算符

##### 概述

在数学中，一个数据x，大于3，小于6，用表达式可以表示为：==3 < x < 6==

在java中，需要把上面的式子先进行拆解，在进行合并表达式

+ 拆解为 ：==x > 3== 和 ==x < 6==
+ 合并为：==x>3 && x < 6==

==&&==就是一个逻辑运算符

**逻辑运算符：**是用来连接关系表达式的运算符，同时，逻辑运算符也可以直接连接布尔类型的常量或变量。

##### 逻辑运算符

| 符号 | 作用     | 说明                                            |
| ---- | -------- | ----------------------------------------------- |
| &    | 逻辑与   | a & b，a和b都是true，结果为true，否则为false    |
| \|   | 逻辑或   | a \| b，a和b任一为true，结果为true，否则为false |
| ^    | 逻辑异或 | a ^ b，a和b结果不同为true，相同为false          |
| !    | 逻辑非   | !a，结果和a的结果相反。                         |

#### 短路逻辑运算符

| 符号 | 作用   | 说明                           |
| ---- | ------ | ------------------------------ |
| &&   | 短路与 | 作用与&相同，但是具有短路效果  |
| \|\| | 短路或 | 作用与\|相同，但是具有短路效果 |

==**注意事项：**==

+ 逻辑与==&==，无论左边真假，右边都会执行

  短路==&&==，如果左边为真，右执行，如果==左边为假，右边不执行==

+ 逻辑或==|==，无论左边真假，右边执行

  短路或==||==，如果左边为假，右边执行，如果==左边为真，右边不执行==

#### 三元运算符

**格式：**关系表单式 ? 表达式1:表达式2

~~~java
a > b ? a:b;
~~~

计算规则：

+ 首先关系计算表达式的值
+ 关系表达式的值为true。表达式1的值为运算结果，否则表达式2为运算结果。

### Scanner使用的基本步骤

+ 导包

  ~~~java
  import java.util.Scanner; // 
  ~~~

+ 创建对象

  ~~~java
  Scanner sc = new Scanner(System.in);
  ~~~

  ~~~java
  int i = sc.nextInt();
  ~~~

~~~java
import java.util.Scanner;

public class ScannerDemo{

        System.out.println("a="+a)
    }
}
~~~

### 分支结构

#### if语句

**格式1**

~~~java
if(关系表达式){
	语句体
}
~~~

> 执行流程：
>
> + 首先计算关系表达式的值
> + 如果关系表达式的值为true则执行语句体否则不执行
> + 继续执行后面的语句



**格式2**

~~~java
if(关系表达式){
	语句体1;
}else{
    语句体2;
}
~~~

> 执行流程：
>

if(关系表达式1){

> 执行流程：

#### switc语句

~~~java
// 格式
switch(表达式){
	case 值1:
		语句体1;
		break;
	case 值2:
		语句体2;
		break;
		....
	default:
    	上面所以条件不为true时触发；
    	break;

~~~

> 格式说明：
>

> + break：表示中断，结束的意思，用来结束switc语句

**注意：**在switch语句中

### 循环结构

==循环结构的组成：==

初始化语句：用于表示循环开启的起始状态，即循环开始的时候是什么样的。

条件判断语句：用于表示循环反复执行的条件，即判断循环能否一直执行下去。

循环体语句：用于表示循环反复执行的内容，即循环反复执行的操作。

条件控制语句：用户表示循环执行中每次变化的内容，即控制循环能否执行下去。

==循环结构对应的语法：==

+ 初始化语句：可以是一条或者多条语句，这些语句可以完成一些初始化操作
+ 条件判断语句：这里使用一个结果值为boolean类型的表达式，这个表达式能决定是否执行循环体，例如：a > 0
+ 循环语句：这里可以是任意语句，这些语句将反复执行
+ 条件控制体语句：这里通常是使用一条语句来修改变量的值，从而达到控制循环是否继续向下执行的效果，常见为i++，i--

#### for循环

~~~java
格式：
    for(初始化语句;条件判断语句;条件控制语句){
        循环体语句;
    }
~~~

>执行流程：
>
>+ 执行初始化语句
>
>+ 执行判断语句，看起结果是true还是false
>
>  如果是false，循环结束
>
>  如果是true，执行循环体语句；
>
>+ 执行条件控制语句
>
>+ 回到第二步执行执行。

#### while循环语句

~~~java
// 基本格式
初始化语句;
 while(条件判断语句){
     循环语句体;
     条件控制语句;
 }

~~~

>执行流程：
>
>1. 执行初始化语句
>
>2. 执行条件判断语句
>
>   ​	如果是false，循环结束
>
>   ​	如果是true，继续执行
>
>3. 执行循环体语句
>
>4. 执行条件控制语句
>
>5. 继续执行第二步
>
>

#### do while循环语句

>基本格式：
>
>​	do {
>
>​		循环语句体;
>
>​	}while(条件判断语句)



>完整格式：
>
>​	do {
>
>​			循环语句体;
>
>​			条件控制语句；
>
>​	}while(条件判断语句)

执行流程：

1. 执行初始化语句

2. 执行循环体语句

3. 执行条件控制语句

4. 执行条件判断语句，看其结果是true还是false

   ​	如果为true，循环结束

   ​	如果为false，执行执行

5. 会到 第二步继续执行操作



**三种循环区别**

+ for逊汗和while循环先判断条件是否成立，然后决定是否执行循环体（先判断后执行）
+ do...while循环先执行一次循环体，然后判断条件是否成立，是否执行循环体（先执行后判断）

#### 跳转控制语句

+ continue 用在循环中，基于条件控制，跳过某次循环体内执行，继续执行下一次的循环体
+ break 用在循环中，基于条件控制，终止循环体内容的执行，也就是结束当前整个循环

### 初始化

java中的数组必须先初始化才能使用。

所谓初始化，就是为数组中的数组元素分配内存空间，并为每个数组元素赋值。

**数组初始化化的方式：**

+ 动态初始化
+ 静态初始化



**动态初始化：**

动态初始化：初始化时指定数组长度，由系统为数组分配初始值

+ 格式：数组类型[] 变量命 = new 数据类型[数组长度]

  ~~~java
  int[] arr = new int[3]; 
  ~~~





**静态初始化**

静态初始化：初始化时指定数组元素的初始值，由系统决定数组长度

+ 格式： 数据类型[] 变量名 = new 数据类型[]{数值1,数值2,数值3,....}
+ 示例： int[] arr = new int[]{1,2,13,4}
+ 简化格式：数据类型[] 变量名 = ｛数据1,数据2，数据3，...｝
+ 示例： int[] arr = {1,2,3}



**数组元素访问**

+ 数组变量访问方式
+ 格式：数组命
+ 数组内部保存的数据访问方式
+ 格式：数组命[索引]

**索引**

+ 索引是数组中的编号方式
+ 作用：索引是用于访问数组中的数据使用，数组命[索引]等同于变量名，是一张特殊的变量名

**索引特征**

1. 从0开始
2. 连续
3. 索引逐一增加，没几次加一



eg：

~~~java
// 定义数组
int arr = new int[3];
// 输出数组元素
System.out.prontln(arr[1]);
System.out.prontln(arr[2]);
System.out.prontln(arr[3]);
~~~





### 内存分配

java程序在运行是，需要在内存中分配控件，为了提高运算速来，就对空间进行不同区域的划分，因为每一片区域都有特定的处理数据方式和内存管理方式。

+ 栈内存：存储局部变量

  定义在方法总的变量，例如：int[] arr,使用完毕立即消失

+ 堆内存：储存new出来的内容（实体、对象）

  数组在初始化是，会为存储空间添加默认值

  >整数：0
  >
  >浮点数：0.0
  >
  >布尔：false
  >
  >字符：空字符
  >
  >应用数据类型：null
  >
  >每一个new出来的对象都有一个地址值，使用完毕，会在垃圾回收器空闲时被收回。
  >
  >
  

当两个数据指向同一个堆内存时，任何一个发生变化，两个也跟着发生变化

~~~java
int[] arr = new int[3];
arr[0] =100;
arr[1] = 200;

int[] arr2 = arr;
arr[0] = 300
// 此时数组arr[0]和arr2[0]都为300
~~~
